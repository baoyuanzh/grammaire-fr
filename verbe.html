动词

动词的句法结构

动词有三类

  1. 单独使用的动词，即不带宾语
  2. 后跟表语的动词
  3. 后跟一个或两个宾语的动词；名词，不定式，从句亦可充当宾语

单独使用的动词

一般动词 rire pleurer souffrir rougir pâlir
dormir rester naître vivre mourir
位移动词 marcher courir arriver partir venir
entrer sortir avancer reculer

后跟表语的动词

主要是动词 être
以及动词 paraître sembler avoir devenir
某些动词 vivre mourir tomber rester
这些动词的直接宾语可以有表语 croire rendre trouver nommer

后跟一个或两个宾语的动词

宾语有两种

  1. 直接宾语 (COD)
  2. 间接宾语 (COI) -- 动词通过介词 de 或者 à 间接作用于宾语，也可能是其他介词，如 sur

后跟不定式宾语的动词

动词+不定式 aimer pouvoir vouloir savoir espérer
penser
动词+介词+不定式 介词 de essayer avoir besoin avoir envie oublir accepter
介词 à tenir commencer se mettre penser chercher
réussir

后面跟有双宾语的动词

动词+直接宾语+间接宾语(à quelqu'un) apporter achter demander montrer envoyer
动词+间接宾语(à qqn)+de+不定式 dire permettre promettre demander écrire
动词+直接宾语(qqn)+à+不定式 autoriser forcer obliger
动词+直接宾语(qqn)+de+不定式 accuser empêcher persuader convaincre
动词+直接宾语(qqn)+de+名词 accuser avertir féliciter informer excuser

后跟一个宾语从句

宾语从句有两类

  1. 一般的宾语从句
  2. 间接问语作宾语从句

多重结构的的动词

  1. 某些动词可以是及物的也可以是不及物的，这种转换有可能引起意义的变化
  2. 某些动词可能有几种结构，因此词义有变化

比如

  • Servir
    • servir qqch à qqn
    • servir à + 不定式
    • se servir
    • se servir de
  • Manquer
    • manquer de qqch
    • manquer qqch
    • verbe impersonnel
    • manquer à qqn
  • Faire
    • faire + 直接宾语
    • faire + 不定式
      • 主动意义
      • 被动意义
    • 动词 faire 也可以代替一个动词词组

  1. 主动式

半助动词

半助动词后面总跟有一个不定式，有如下几种半助动词

它们用于指出时间

  1. 将来时
    1. aller + 不定式：最近将来时
    2. Devoir + 不定式
  2. 过去时
    1. venir de + 不定式：最近过去时

它们指出动词的一个体

体指的是一个动作的开始，进行或完成。

开始 se mettre à commencer à/de être sur le point de
进行 être en train de continuer à
完成 finir de cesser de arrêter de

它们具有语式意义

即它们指出讲述着的态度，想法

  1. Devoir
    1. 必须
    2. 可能
  2. Pouvoir
    1. 做某事的能力，能，能够
    2. 允许，许可
    3. 礼貌
    4. 偶然性
  3. Savoir
    1. 指熟悉规则，技能
  4. Faire
    1. 是某个动作的原因
  5. Laisser
    1. 不阻止某个动作

动词和主语的配合

动词要和主语配合，主语可以是名词也可以是代词。

一般规则

配合要考虑两方面：

  1. 人称
  2. 数

需要注意的是：

  1. 当主语是关系代词 qui 时，动词要和现行词配合
  2. 在疑问句或关系从句中，主语可能在动词的后面

特殊情况

主语是不同的人称

  1. 第一人称 + 第二人称 = 我们
  2. 第三人称 + 第一人称 = 我们
  3. 第三人称 + 第二人称 = 你们

主语用 ou 或者 ni ... ni ... 并列

  1. Ou : 根据句子的意义决定动词用复数或单数
  2. Ni ... ni ... : 动词用复数或单数均可

主语是一个集体名词

  1. 如果主语是一个没有补语的集体名词，动词用单数。如 peuple, foule, groupe, équipe 等
  2. 当集体名词后面跟有一个复数的补语时，动词用单数（与集体名词配合）或复数（与复数补语配合）

Beaucoup, peu, trop, assez, combien, la plupart, xx%(etc) + de + 复数名词

动词用复数

Le seul qui, le premier qui, le dernier qui, etc

可以使动词作如下配合：

  1. 和主语动词的主语配合
  2. 和 le seul, le premier, le dernier 配合

被动式或被动态

被动式与主动式从两个不同的观点介绍一个相同的动作。

被动态的构成

从主动式过渡到被动式

这个过渡会引起一些变化

  1. 主动式动词的直接宾语 >>>>> 被动式动词的主语。
  2. 主动式动词的主语 >>>>> 被动式动词的施动者补语，施动者补语由 par 引导。
  3. 主动式的动词 >>>>> être + 过去分词，助动词 etre 的时态与主动式动词相同。

只有直接及物动词才能变成被动式。

当施动者补语是人称代词时，一般不用被动式。

被动式的动词变位

动词的时态由助动词指出。过去分词总是和主语搭配

  1. 直陈式
    1. 现在时
    2. 将来时
    3. 先将来时
    4. 复合过去式
    5. 未完成过去时
    6. 愈过去时
    7. 简单过去时
    8. 先过去时
  2. 条件式
    1. 现在时
    2. 过去时
  3. 虚拟式
    1. 现在时
    2. 过去时

副词一般放在助动词和过去分词之间。

被动式的用法

主动式与被动式比较

它们并非绝对相等。

  1. 句子一般强调的是动词主语，因此根据需要强调的内容选择主动式或被动式。
  2. 当施动者不重要或不明显时，被动式常用语这样的无施动者补语的句子，以强调结果。
  3. 被动式也具有描写意义。在这种情况下，过去分词变成品质形容词。

施动者补语：用 par 还是用 de ？

大部分的动词在结构上用介词 par 。然而，在它们之中有几个用介词 de 更好。

  1. 表示描写的的动词，特别是当施动者是非生物时：
    1. être accompagné
    2. être composé
    3. être couvert
    4. être décoré
    5. être entouré
    6. être fait
    7. être garni
    8. être orné
    9. être orné
    10. être planté
    11. être précédé
    12. être rempli
  2. 表示情感的动词
    1. être admiré
    2. être adoré
    3. être aimé
    4. être apprécié
    5. être craint
    6. être estimé
    7. être haï
    8. être méprisé
    9. être redouté
    10. être respecté
  3. 其他动词用作本义时用 par ，用作转义时用 de 。

表达被动意义的其他方法

被动式意义的代词式动词

se + 动词

这种句法结构是常见的。主语常为非生物动词，施动者不必指出。

动词 faire + 不定式

这种句法结构具有被动意义。

Se faire 和 se laisser + 不定式

主语总是生物动词。施动者者并非总要表达出来。

  1. se faire 表示主语有一定的责任。
  2. se laisser 强调主语的被动性。

无人称被动句

有些被动式动词可以用作无人称形式，特别在行政用语和报业用于中。这些用法能够缓和主语的强硬语气。

  1. Il a été décidé de + ...
  2. Il est recommandé de + ...
  3. Il a été prouvé que + ...

代词式

代词式动词是前有一个和主语相同人称的自反代词的动词。

  1. 在有些情况下，自反代词是动词的宾语。
  2. 在另一些情况下，自反代词没有语法功能，仅仅是动词的一个构成部分而已。

代词式动词的构成

动词前总有和主语相同人称的自反代词。

je me
tu te
il / elle se
nous nous
vous vous
ils / elles se

在复合时态中，助动词用 être 。

je me suis
tu t'es
il / elle s'est
nous nous sommes
vous vous êtes
ils / elles se sont

代词式动词的种类

代词式动词有多个种类，同一个动词可能属于多个种类。

自反意义的代词式动词

  1. 主语为自己施动或者主语是动作的承受着。
  2. 当主语对自己身体的一部分实施某个动作或主语的身体的一部分遭受某一动作时，常用代词式动词。

相互意义的的代词式动词

几个主体相互间施行某个动作，因而主语必定是复数，这种情况的动词有：

  1. se téléphoner
  2. se parler
  3. se rencontrer
  4. se lancer
  5. se donner
  6. s'envoyer

被动意义的代词式动词

动词表示的动作非主语所为

绝对意义的代词式动词

这些动词只有代词式：

  1. se souvenir
  2. s'en aller
  3. s'abstenir
  4. s'évanouir
  5. s'absenter
  6. s'envoler
  7. s'enfuir
  8. s'emparer
  9. se méfier

在代词式中另有他义的动词

passer se passer

attendre s'attendre

属于多个种类的动词

  1. apercevoir
  2. entendre

过去分词的配合

自反意义和相互意义的动词

  1. 过去分词和助动词前作直接宾语的代词配合，直接宾语可能是人称代词或关系代词。
  2. 当直接宾语是放在动词后的名词时，过去粉刺不配合。
  3. 过去分词不和间接宾语配合。

绝对意义的代词式动词，另有他义和被动意义的代词式动词

过去分词和主语配合。

无人称结构

无人称动词是用于第三人称单数的动词，主语 il 不表示任何意义。

天气情况的动词

  1. Il pleut
  2. Il neige
  3. Il gèle
  4. Il fait + 形容词或名词
    1. Il fait beau, froid, bon, doux, chaud, etc
    2. Il fait un temps splendid, il fait un froid de canard, etc.
    3. Il fait 15 ˆC.

Il est + 时间词语

为了指出某一时刻

  1. Il est midi, minuit, tôt, tard, etc.

  1. Il est temps +

  2. Il est l'heure +

Il y a + 名词或代词

  1. 为了指出一个人或事的存在

Il + 后随“实质主语”的动词

“实质主语”是放在动词后的名词，无人称结构不重视主语，而强调动词

Il + 被动式或代词式动词

这个用法在行政用语和报业用语中是常见的。

语式，时态，体

一个动词的形式随 语式、 时态 和 体 的变化而变化。

语式

动词的语式使说话人能够表达其所说言语的态度。

有四个人称语式

直陈式 表示一个确实发生的动作，它在时间上“指出”动作发生在某一时刻
虚拟式 表示对某一个动作的某种判断（愿望，怀疑，意志等）
条件式 表示一个假定发生的动作
命令式 表示一个命令

有两个非人称语式，其形式不取决于主语

  1. 不定式
    1. 不定式现在时
    2. 不定式过去时
  2. 分词式
    1. 现在分词
    2. 过去分词

时态和体

时态 是确定一个动作发生的时间实在过去、现在还是将来。
体 是表示动作进行的方式：开始，持续，完成。

同一个时态可能用来按照时间先后的顺序确定一个动作发生的时间和具有体的意义。

每一个语式包含简单时态和符合时态

简单时态 由词干和词尾构成。
复合时态 由助动词和过去分词构成。

某些动词具有体的意义

  1. commencer à
  2. finir de
  3. continuer à
  4. devoir
  5. être sur le point de
  6. être en train de

直陈式

直陈式是表示一个确实发生的动作的语式。它包括许多时态（简单时态与复合时态）。这些时态能够明确地确定一个动作的发生的时间是在过去、现在或将来。

直陈式现在时

构成

  1. 第一组动词，有1个词干，以 -er 结尾：词干+ -e, -es, -e, -ons, -ez, -ent
  2. 第二组动词，有2个词干，以 -ir 结尾：词干+ -s, -s, -t, -ons, -ez, -ent
  3. 第三组动词，有1个，2个或3个词干，以 -ir, -oir, -re 结尾：
    1. 有1个词干：词干+ -e, -es, -e, -ons, -ez, -ent
    2. 有2个词干：词干+ -s, -s, -t/-d, -ons, -ez, -ent
    3. 有3个词干：词干+ -s/-x, -s/-x, -t, -ons, -ez, -ent

用法

  1. 现在时把一个动作发生的时间确定在说话时刻。它表示正在完成中的动作。
  2. 它用于描写。
  3. 现在时没有明确的时间界限，但伴有时间的标志，它能表示：
    1. 一个在过去已经开始的动作。
    2. 一个即将实现的动作。
    3. 一个习惯性、重复性的动作。
  4. 它用于一篇分析性文章，一部电影，一部作品的梗概或结论。
  5. 它用来表示一个永恒的真理，一个谚语。
  6. 它用在以 si 引导的假设从句中。
  7. 它能具有命令式的意义。

直陈式未完成过去时

构成

除了 être 外所有动词的直陈式未完成过去时都是规则的！

规则：直陈式现在时复数第一人称的词干+ -ais, -ais, -ait, -ions, -iez, -aient

例外 être ：

j'étais tu étais il/elle était
nous étions vous étiez ils/elles étaient

用法

直陈式未完成过去时和直陈式现在时一样，表示一个正在完成的动作。它在时间上没有明确的界限。

  1. 它用于描写、品论、说明等等。
  2. 它用来表示一个习惯性的动作，这时它总是伴有一个时间标志。
  3. 直陈式未完成过去时与和连词 si 连用时，并不是一个过去时态。它表示对一个现在的动作的假设或是表示该动作是不真实的。
  4. 它用于礼貌用于，使语气婉转。

直陈式复合过去时

构成

用 avoir 或 être 的直陈式现在时作助动词 + 过去分词

用法

直陈式复合过去时表示一个近期的或遥远的过去一定时刻完成的动作。

  1. 点动作
  2. 连续的事件
  3. 重复性
  4. 有限的延续
  5. 它表示一个延续到现在时刻已经完成的动作。

直陈式复合过去时是用于谈话、书信的时态，这就是说它是日常交际的时态，不同于只用于书面语的直陈式简单过去时。

直陈式未完成过去时和直陈式复合过去时比较

某情景下要说清楚到底该用这两者中的哪一个常常是困难的。

  1. 有时，两个动作同时出现在一个句子中，一个动作正在完成中 (imparfait) 而另一个动作已完成 (passé composé)
  2. 有时，未完成过去时营造（描写）了一个背景，而另一个动作在这个背景下完成 (passé composé)
  3. 虽然两者都可以表示习惯性的动作，但记住一点，未完成过去时表示动作发生在一个不明确的延续时间里，而复合过去时表示动作发生在一个明确的且有限的延续时间里。
  4. 通常没有任何时间标志的时候用未完成过去时，而有时间标志的时候用复合过去时，注意这个时间标志可能不明显。

最近过去时

构成

用 venir 的现在时或未完成过去时 + de + 不定式

用法

最近过去时用来表示一个不久前完成的动作。

在一个用过去时的具体上下文种，动词 venir 用未完成过去时。

简单过去时

构成

简单过去时的词干在各个人称都是相同的。

  1. 第一组动词：词干 + -ai, -as, -a, -âmes, -âtes, -èrent
  2. 第二组动词：词干 + -is, -is, -it, -îmes, -îtes, -irent
  3. 第三组动词：词干 + -is, -is, -it, -îmes, -îtes, -irent 或 词干 + -us, -us, -ut, -ûmes, -ûtes, -urent

用法

简单过去时和复合过去时（点动作、有限的延续、连续的事件）意义相同，除了后者可以表示一个先于现在时已经完成的动作外。

在一篇用过去时的叙述里，简单过去时和未完成过去时的关系与复合过去时和未完成过去时的关系相同。但简单过去时有别于把动作或行为与说话时刻联系起来的复合过去时，它把动作发生的时刻确定在一个遥远的过去（和现在没有联系）。所以简单过去时用于文学（小说，故事，儿童故事，传记等），尤其用于第三人称。

直陈式愈过去时

直陈式愈过去时是复合时态，具有完成的意义。

构成

用 avoir 或 être 的直陈式未完成过去时 + 过去分词。

用法

愈过去时表示对另一个过去发生的动作而言，先于该动作的一个已经完成的动作。

和连词 si 连用，它表示一个过去的动作是不真实的。

先过去时

构成

用 avoir 和 être 的简单过去时作助动词 + 过去分词

用法

先过去时专门用于书面语。它用于时间从句里。对另一个用简单过去时表示的动作而言，它强调先于该动作的一个刚刚完成的动作。

超复合过去时

构成

用 avoir 和 être 的复合过去时作助动词 + 过去分词

用法

超复合过去时用于时间从句。对另一个用复合过去时表示的动作而言，它强调先于该动作的一个刚刚完成的动作。这是一个不常用的时态。

简单将来时

构成

简单将来时以不定式为词干构成，除了有些第三组动词外，所有动词的词尾都是一样的：-ai, -as, -a, -ons, -ez, -ont

  1. 第一组动词和第二组动词：不定式 + 词尾
  2. 第三组动词：
    1. 以 -re 结尾的动词，去掉 -e + 词尾
    2. 以 -ir 结尾的动词，不定式 + 词尾
    3. 特别词干的动词，个别记忆
    4. 以 -oir 结尾的动词，词干 + r + 词尾

用法

  1. 简单将来时把动作发生的时间确定在近期或遥远的未来。它常常伴有时间标志。
  2. 简单将来时能具有命令式的意义，它缓和了命令语气。

最近将来时

构成

动词 aller 的现在时 + 不定式

用法

  1. 它把动作确定在最近的将来。
  2. 它也表示被确定在遥远的未来确实将发生的动作。
  3. Devoir 和 être sur le point de 是语式半助动词，它们也具有最近将来时的意义。

先将来时

构成

用 avoir 或 être 的简单将来时作助动词 + 过去分词

用法

  1. 先将来时对简单将来时或命令式现在时而言，用来表示在他们之前已经完成的动作。
  2. 它单独使用，则对将来的一定时刻（该时刻一般有明确的时间标志）而言，表示一个将确实发生并且完成的动作。
  3. 先将来时也可以表示一种可能性。

过去的将来时

在一个用过去时的具体上下文中，使用过去的将来时。过去的先将来时和过去的最近将来时要遵守配合的规则。

  1. 过去的简单将来时：与条件式现在时相同
  2. 过去的先将来时：与条件式过去时相同
  3. 过去的最近将来时：动词 aller, devoir 和 être sur le point de 的未完成过去时 + 不定式

直陈式的时态配合

在一个复合句中，当主语的动词用过去时（未完成过去时，复合过去时，简单过去时，愈过去时）时，从句的动词也应当是一个过去的时态。这就是时态配合的规则。

该规则也有例外：

  1. 当主句动词是复合过去时时
  2. 当从句表达的是一个永恒的真理时

虚拟式

虚拟式使说话人能够对一个动作做出判断和解读。它是主观的语式，主要用于从句中。

虚拟式有四个时态：

  1. 两个简单时态：现在时和未完成过去时。
  2. 两个复合时态：复合过去时和愈过去时。

未完成过去时和愈过去时属于雅语或文学语言，并不常用。

虚拟式的构成

虚拟式现在时

  1. 直陈式现在时复数第三人称 + -e, -es, -e, -ions, -iez, -ent
  2. 有些第三组动词的虚拟式是在两个词干的基础上构成的。
    1. 四个人称 je, tu, il, ils 是在直陈式第三人称复数的词干基础上构成的。
    2. 两个人称 nous, vous 是在第一人称复数的词干基础上构成的。

虚拟式过去时

用 avoir 或 être 的虚拟式现在时作助动词 + 过去分词

虚拟式未完成过去时

在直陈式简单过去时的基础上 + -sse, -sses, -t, -ssions, -ssiez, -ssent

虚拟式愈过去时

用 avoir 和 être 的虚拟式未完成过去时作助动词 + 过去分词

虚拟式的用法

  1. 虚拟式主要用于从句中
    1. 补语从句
    2. 时间从句
    3. 目的从句
    4. 对立从句
    5. 条件从句
    6. 原因从句
    7. 后果从句
    8. 关系从句
  2. 虚拟式也可以单独使用：用 que 引导的第三人称单数的虚拟式具有命令式的意义。在固定熟语中也有该用法，其前面的 que 可以省略。

虚拟式时态的意义

虚拟式现在时意义

  1. 同时性：从句动词用虚拟式现在时，对主句动词而言具有同时性，这就是说，主句和从句的动作被确定在同一时刻发生。其实在当代法语中，只有在特殊情况下才遵循虚拟式的时态配合规则。即使主句动词用过去时，也不用虚拟式未完成过去时，而用虚拟式现在时。
  2. 后时性：虚拟式现在时对主句动词而言，可以具有将来的意义。这是，主句动作可能用现在时，将来时，条件式或过去时。从句所表达的动作发生在主句所表达的动作之后。

虚拟式过去时的意义

  1. 先时性：从句动词用虚拟式过去时，当从句所表达的动作发生在主句动作之前时。这时，主句动词可以用现在时，过去时，将来时或条件式。
  2. 动作的完成：对一个确定在将来的时间界限而言（而且通常该时间界限由时间标志指出），虚拟式过去时也可以表达一个已经完成的动作。主句动词可以用现在时，过去时，将来时或条件式。

未完成过去时和愈过去时（雅语）

这两个时态使我们能够遵循时态配合规则：当主句动词用过去时时，用未完成过去时表示对主句动词而言的从句动作的同时性或后时性；用愈过去时则表示从句动作的先时性和动作已经完成。但是，如今，这样的时态配合只能在雅语中见到了。

条件式

条件式表示一种可能，不真实或想像的语式。它也具有过去将来时的意义。

条件式的构成

条件式有两个时态：现在时和过去时。

现在时

将来时的词干 + 未完成过去时的词尾 -ais, -ais, -ait, -ions, -iez, -aient

过去时

用 avoir 或 être 的条件式现在时作助动词 + 过去分词。

条件式的用法

条件式作为语式

它可以表示：

  1. 愿望、希望：条件式现在时
  2. 遗憾：条件式过去时
  3. 未经证实的消息，假设
  4. 一个想象的动作
  5. 缓和语气：礼貌的或暗示的条件式
  6. 惊讶
  7. 建议，用于下列动词的条件式现在时：
    1. devoir + 不定式
    2. faire mieux de + 不定式
    3. valoir mieux + 不定式 / que + 虚拟式
  8. 指责，用于下列动词的条件式过去时：
    1. pouvoir + 不定式
    2. devoir + 不定式
    3. valoir mieux + 不定式 / que + 虚拟式

此外，条件式

  1. 相对于直陈式而言，还带有可能性的色彩
  2. 也用于由 si 引导的从句有关联的主句中
  3. 在连词 au cas où, quand bien même 后

条件式作为时态

当主句动词用一个过去的时态（复合过去时，未完成过去时，愈过去时，简单过去时）时，它具有将来时的意义。我们应当遵守时态配合的规则。

  1. 条件式现在时 = 过去的将来
  2. 条件式过去时 = 过去的先将来时

命令式

命令式是表示种种有细微差别的命令：禁止，建议，请求等等的语式。

命令式的构成

命令式只有三个人称：

  1. 第二人称单数
  2. 第二人称复数
  3. 第一人称复数

对第三人称单数和复数的命令句用虚拟式

命令式有两个时态：

  1. 现在时
  2. 过去时

现在时

形式为这三个人称的直陈式现在时的形式，注意用感叹号，此外以 -es 结尾的第二人称单数的动词去掉 s 词尾为 -e 。aller 为特殊情况，tu vas 改为 va!

有三个动词的命令式不规则：

  1. avoir: aie, ayons, ayez
  2. être: sois, soyons, soyez
  3. savoir: sache, sachons, sachez

代词式命令句，动词和代词之间有连词符

过去时

它由动词 avoir 或 être 的命令式现在时作助动词 + 过去分词构成。

命令式的用法

现在时

  1. 它根据具体的上下文表示命令：禁止、希望、请求
  2. 它具有最近的或较远的将来时意义
  3. vouloir 的命令式只有一个形式：veuillez
    1. 它是礼貌的命令式
    2. 它在书面语里用于正式的信函

过去时

它不常用，指出在将来的一个确定的时刻之前应该完成的动作，一般有时间标志的提示。
